# HOW TO USE

	Step 1. Place your chart (in .json format) in 'json' folder.
	Step 2. Open the application and follow the instructions.
		2.1. Type your chart's name in the prompt (without .json).
		2.2. Type what note type it should use instead of 'noteData > 7'.
	Step 3. The converted chart should be in 'outputs' folder.
